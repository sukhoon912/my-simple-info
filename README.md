My Simple Info 👤

자기소개와 기술 스택, 연락처 정보를 정리한 개인 프로필 웹 페이지입니다.
간결한 레이아웃으로 핵심 정보만 직관적으로 보여주는 정적 웹 프로젝트입니다.

기능

✅ 자기소개(About Me) 섹션
✅ 기술 스택 소개 (HTML / CSS / JavaScript)
✅ 연락처 정보 제공
✅ 심플하고 직관적인 UI
✅ 반응형 디자인

사용 방법

웹 페이지에 접속하면
자기소개, 기술 스택, 연락처 정보를 한눈에 확인할 수 있습니다.

실행 방법

index.html 파일을 브라우저에서 직접 열거나,
GitHub Pages 배포 링크로 접속하면 됩니다.
